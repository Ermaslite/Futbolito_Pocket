Este trabajo consiste en simular el clásico jueguito de futbolito de bolsillo dentro de una aplicación Android, utilizando Jetpack Compose como framework principal.

La dinámica del juego se basa en:

Movimiento de la pelota: controlado mediante los sensores del dispositivo (acelerómetro/giroscopio), lo que permite que el usuario incline el teléfono para guiar el balon por la cancha.

Campo de juego: la pelota no podrá salir de los límites; únicamente podrá rebotar en las paredes dentro de la cancha.

Marcador dinámico: cada vez que la pelota entre en la portería, se sumará un gol al marcador y se reproducirá un efecto de sonido para dar mayor realismo
